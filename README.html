poisonfrogs

poisonfrogs is a extensive collection of colour palettes inspired in species and morphs of Neotropical poison frogs. And more and more

Installation

You can install the development version of poisonfrogs from GitHub with:

# install.packages("pak")
pak::pak("laurenoconnelllab/poisonfrogs")

or with:

install.packages("poisonfrogs")
#> Warning: package 'poisonfrogs' is not available for this version of R
#> 
#> A version of this package for your version of R might be available elsewhere,
#> see the ideas at
#> https://cran.r-project.org/doc/manuals/r-patched/R-admin.html#Installing-packages
